# Design Backlog

> UX debt tracker. Separate from INSIGHTS.md (which is for cross-pioneer learnings, not debt).
> Every task that touches a visible component and deviates from Visual DNA must log an entry here.

## Rule

Until Visual DNA coverage reaches 80%, every task that touches a visible component must include a UX debt note in this file. This rule expires when @copilot or @ui has closed 80% of entries below.

## Format

```

- **Created by:** @{pioneer}
- **Date:** {YYYY-MM-DD}
- **Component:** {file path}
- **Deviation:** {what deviates from Visual DNA}
- **Status:** open | closed
- **Closed by:** @{pioneer} in {PR/commit}
```
