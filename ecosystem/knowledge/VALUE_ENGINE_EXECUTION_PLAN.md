## BLOCK 2 — TOTAL COVERAGE DOMAIN MAP

```
DOMAIN A — CREATION
  Info products:    book · guide · report · framework doc · research pack
  Teaching:         course · cohort · curriculum · workshop · masterclass
  Tools:            SaaS · template · toolkit · workflow pack
  Services:         consulting · done-for-you · advisory · retainer · fractional
  Authority:        membership · newsletter · private community · mastermind
  Assets:           proprietary frameworks · named methodologies · intellectual models
  Leverage:         1→many teaching > 1→many media > 1→many tools > network effects

DOMAIN B — MARKET / ACQUISITION
  Attention:        category distinctness · hook precision · entry point clarity
  Audience:         trust accumulation · repeat signal · community formation
  Channel:          platform fit · leverage points · signal-to-noise
  Positioning:      category ownership · contrast framing · competitor displacement
  Trust:            consistency signals · proof structures · authority indicators
  Niche:            underserved segment · signal language · territory claim
  Law:              distribution without positioning = noise
                    positioning without distribution = irrelevance

DOMAIN C — MONETIZATION
  Value ladder:     free → entry → core → premium → sovereign
  Offer design:     promise · mechanism · proof · anchor · risk reversal · boundary
  Pricing:          luxury / premium / value / access positioning tiers
  Continuity:       subscription to access / service / transformation / community
  Conversion law:   precision of match, not persuasion mechanics

DOMAIN D — EXECUTION
  Systems:          SOP · workflow · execution map · asset pipeline
  Ladders:          solo → systems-dependent → semi-autonomous
  Decision trees:   if/then flows that eliminate judgment overhead
  Law:              execution with system = leverage
                    execution without system = non-transferable skill

DOMAIN E — TEACHING
  Curriculum:       what · what order · what dependencies
  Skill ladders:    beginner → practitioner → expert (observable proof at each)
  Transformation:   before state → after state · measurable by learner
  Law:              teaching without behavioral change = entertainment

DOMAIN F — VALIDATION
  Offer:            V-01 — real pain + willingness to exchange
  Positioning:      V-02 — message lands without explanation
  Pricing:          V-03 — price reflects perceived value
  Channel:          V-04 — path reaches right person efficiently
  Traction:         V-05 — if X then Y within Z
  Prototype:        V-06 — minimum version delivers without creator
  Benchmark:        V-07 — measurable superiority on defined dimension
  Law:              validation before commitment = calculated risk
                    commitment before validation = gambling
```
