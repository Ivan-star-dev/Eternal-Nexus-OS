## PERGUNTA QUE ESTE CAPÍTULO RESOLVE
O que existe à volta da corporação e como se relaciona com ela?

## CONTEÚDO PRINCIPAL
- Mapa do ecossistema: parceiros · fornecedores · integrações · público
- Tipos de relação: fornecedor / parceiro / cliente / regulador / aliado
- A borda pública: onde RUBERRA toca o mundo exterior
- O princípio: o ecossistema serve a soberania, não o contrário

## HIERARQUIA VISUAL
MACRO → ecosystem map: RUBERRA no centro, relações em camadas
MESO → tipos de relação por camada
MICRO → exemplos concretos de cada tipo

## ELEMENTOS VISUAIS
- Concentric circles: RUBERRA → parceiros próximos → ecossistema → mundo
- Edge labels por tipo de relação
- Color coding: gold para aliados estratégicos / branco para operacionais
- Borda pública destacada

## TOM
Estratégico. O ecossistema é mapeado como território, não como lista de contactos.
