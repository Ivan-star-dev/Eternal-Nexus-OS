## PERGUNTA QUE ESTE CAPÍTULO RESOLVE

## CONTEÚDO PRINCIPAL

### A. Naming Standards
- Entidades canônicas: MAIÚSCULAS
- Instâncias: camelCase
- IDs: monospace com prefixo
- Ficheiros: CAPS_WITH_UNDERSCORES.md

### B. Visual Grammar
- Cores: gold (#D4AF37) · teal (hsl 172) · black (#030303) · white
- Tipografia: serif (autoridade) · mono (dados) · sans (corpo)
- Espaçamento: generoso, respirável, nunca comprimido
- Hierarquia: sempre MACRO → MESO → MICRO

### C. Template Canônicos
- HANDOFF block template
- BASTION task block template
- SITUATION REPORT template
- CHANGELOG entry template

### D. Commit Standards
- Formato: `tipo(escopo): descrição`
- Tipos: feat / fix / ops / protocol / docs
- TS 0 obrigatório antes de qualquer commit
