Cristalizar, num único artefacto canónico de referência, todo o conhecimento operacionalmente válido que emergiu do debate em cascata — separando o que fica como lei do que adapta, e garantindo que nada brilhante se perde por falta de formalização.

## 4. CORE LAWS

```
CRC-LAW-001  O produto é organismo soberano — não site, não app, não dashboard.
CRC-LAW-002  A Mãe precede os filhos — identidade-mãe nunca diluída por face filha.
CRC-LAW-003  A Trindade é imutável — Heaven Lab · Bridge Nova · Nexus Cria.
CRC-LAW-004  O runtime adapta o ambiente à mente — não o utilizador monta o sistema.
CRC-LAW-005  A sessão não morre — tem entrada, lock, materialização, persistência.
CRC-LAW-006  Toda sessão gera fruto real — sem fruto, a sessão falhou.
CRC-LAW-007  O eixo moral mede pelo alinhamento com a vida — não por dogma, não por feeling.
CRC-LAW-008  A corporação é backlane paralela — não bloqueia nem roubar prioridade central.
CRC-LAW-009  A sequência V3→V4→V5 é inviolável — gate anterior fecha antes de próximo abrir.
CRC-LAW-010  O sistema amplifica agência — a pessoa sai mais capaz, não mais dependente.
CRC-LAW-011  A costura é invisível — o utilizador nunca sente plugin-zoo.
CRC-LAW-012  Complexidade → ordem — não por simplificação burra, mas por estrutura e discernimento.
```

## 9. ANTI_DRIFT_RULES

```
DRIFT-1: Produto descrito como "site" ou "app" → corrigir para "organismo soberano"
DRIFT-2: Face filha com identidade própria conflituante com mãe → mãe prevalece
DRIFT-3: Sessão sem fruto → sessão considerada falha · redesenhar
DRIFT-4: Eixo moral com dogma ou sentimentalismo → calibrar pelo alinhamento com a vida
DRIFT-5: V5 aberta antes de V3+V4 → STOP imediato · comunicar owner
DRIFT-6: Corporação bloqueando prioridade central → realocar para backlane
DRIFT-7: Plugin-zoo visível ao utilizador → costura invisível é requisito
DRIFT-8: Ambiente não se adapta → utilizador não deve montar o sistema manualmente
DRIFT-9: Complexidade sem discernimento → estrutura · visualização · evidência
DRIFT-10: Capacidade não amplifica agência → redesenhar até amplificar
```
