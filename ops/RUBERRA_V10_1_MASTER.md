## PORTAL LAWS

### Creation Lab
- reduce distance between imagination and execution
- open, calm, premium, spatially intelligent
- supports ideation, structuring, prototyping, building
- stable core work bay, concept surfaces, prototype pods, tool spine, memory wall
- no clutter, no floating chaos, no decorative tech

### Learning School
- transform confusion into progressive understanding
- welcoming, advanced, clear, confidence-building
- guided learning hall, modular stations, demonstration zones, skill path, reflection niche
- adaptive guidance, no infantilized edtech feeling, no overload

### Simulation Hub
- transform hypotheses into testable reality
- modular, systemic, responsive, controlled
- scenario bays, live model surfaces, dynamic response objects, system map layers
- emphasizes feedback, experimentation, validation, world response

### Deep Focus Chamber
- reduce all non-essential cognitive noise
- minimal, elegant, slow, concentrated, emotionally quiet
- strongest density caps, lowest distraction, highest task immersion
- no spectacle, no extra signals

### Memory Archive / Progress Nexus
- make progress visible, legible, and emotionally real
- preserve artifacts, milestones, routes, project continuity, identity traces
- gallery/archive/constellation logic
- memory must feel alive, not like file storage

### Collaboration Atrium
- support clean multi-presence and co-creation
- open but disciplined, shared surfaces, synchronized context, low social noise
- collaboration without clutter or breakdown of identity laws
